# Inventory System


## Getting Started

First, install depedency

```bash
npm install
```

Second, run the Next.js frontend server
```bash
npm run dev
```

## Backend Repository
You can visit [https://github.com/mfitrie/backend-inventory](https://github.com/mfitrie/backend-inventory) for the backend repository.
